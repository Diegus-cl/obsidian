## Enable hover only on non-touch devices 

``` css
@media (hover: hover) and (pointer: fine) {
	/* styles with hover tag here */
}
```


## Check if the user wants reduce motion

```css
@media (prefers-reduced-motion) {
	/* disable animations
	 .class {
	   animation: none;
    */
}
```
