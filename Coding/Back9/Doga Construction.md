Folder structure 

-   src
    -   components
        -   layout.js --- component inside components folder, more than one page will use this component
        -   templates --- components to a specific page
            -   home -- homePage
                -   article.js -- Article Component
        -   ui --- styled components, ex: Button, Title, Link
    -   pages
    -   styles
-   public
